# Sinotech Professionalism Checker
# Исследование корпуса китайской IT-документации: возможности анализа и автоматизации

Данный проект представляет собой исследовательский прототип, созданный в рамках ДПО “Компьютерная лингвистика” для демонстрации преимуществ корпусного анализа при работе с техническими текстами на китайском языке. Инструмент ориентирован на IT-специалистов и переводчиков, работающих с китайским сегментом разработки, и помогает понять стандарты профессионального технического стиля.

## Цель проекта
Основная задача работы — наглядно показать, что статистический анализ данных позволяет выявить устойчивые профессиональные паттерны, которые сложно обнаружить при обычном изучении языка.

## Аналитические возможности
В проекте реализован полный цикл обработки данных, результаты которого представлены в виде графиков и статистических метрик:

1. **Сравнение лексического богатства.** С помощью индекса TTR  математически подтверждается, что профессиональная документация на 40-50% менее лексически разнообразна, чем обычные README-файлы.
2. **Выявление терминологического ядра.** Визуализация наиболее частотных пар существительных позволяет увидеть «скелет» китайской IT-терминологии.
3. **Анализ глагольной сочетаемости.** Графики глагольно-именных конструкций показывают, какие именно действия носители языка совершают над техническими объектами (например, как правильно «развернуть модель» или «настроить параметры»).

## Дополнительный функционал: Интеллектуальная проверка
В качестве демонстрации потенциала проекта в код включен модуль Smart Checker. Это экспериментальная функция, которая показывает, как собранная статистика может помогать пользователю в реальном времени:
* Система находит в тексте пользователя сочетания, которые редко встречаются в профессиональной среде.
* На основе данных из эталонного корпуса (PaddleNLP) программа предлагает более употребимые варианты глаголов для конкретных IT-терминов.

Эта функция не является законченным редактором, а служит примером того, как на базе собранного корпуса можно построить полноценный ИИ-ассистент для технических писателей.

## Технологический стек
* GitHub REST API, spaCy (модель zh_core_web_sm), NLTK, Matplotlib.

## Инструкция по запуску
1. Откройте файл Sinotech_Professionalism_Checker.ipynb в среде Google Colab.
2. Запустите все ячейки по порядку.
3. Изучите графики в разделе визуализации для понимания структуры китайского IT-языка.
4. В финальной части ноутбука можно протестировать работу статистического анализатора на собственном фрагменте текста.

## Перспективы развития
Проведенное исследование подтверждает, что методы корпусной лингвистики значительно упрощают преодоление языкового барьера в узкоспециализированных областях. В будущем проект может быть расширен за счет использования векторных представлений слов для поиска синонимов и интеграции в профессиональные системы автоматизированного перевода.
